User Manual

Home

Use the home page to navigate our web application. The main navigation consists of schedule search, documentation, and about.

Add/Edit

Use the three functions provided to wither add, edit, or delete a movie to the database. When adding or editing a movie the following information is necissary for identification: movie title, movie director, release year, and the country of origin. When deleting a movie all that is necissary is the movie title. When finished filling the fields, click or tap the button below the section to submit your changes.

Developer Manual

How to install application and all dependencies

1. Clone this repository through Github Desktop or through Terminal

2. Open repository in VSCode Terminal or Terminal application

3. type `npm install` into terminal window and run.

4. The application should now be set to run.

How to run application on a server

1. Open repository in VSCode terminal or Terminal application

2. Run npm start. There should be no errors.

3. In a web browser, go to url: http://localhost:3000/

To run tests for software

The are no prewritten tests in the source repository, but you can use Cypress to run your own written tests.

1. Open two terminals and make sure you are in the main project directory

2. In the first terminal, run npm start

3. In the second terminal run npm test

Potential Bugs:

Given the amount of data extracted from the Internet Movie Database, data cleaning has a high probability of being imperfect. As a side effect, some movies may show flaws that automated data cleaning procedures were unable to identify. Future database modifications may need to be implemented.

Future Development:

Incorporating new movies that have released after the database was extracted would help keep the website current. Adding new columns to the data from other sources, such as Rotten Tomatoes or MetaCritic, could be a better gague upon audience reception, bolstering the financial success. Lastly, adding more options to filter through movies, perhaps with genre and actors, would be a beneficial approach to helping a user find what they are looking for.
